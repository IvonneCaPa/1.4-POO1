Ejercicios Nivel 1

Ejercicio 1 Crea una clase Employee, define como atributos su nombre y sueldo. Definir un método initialize que reciba como parámetros el nombre y sueldo. Plantear un segundo método print que imprima el nombre y un mensaje si debe pagar o no impuestos (si el sueldo supera 6000, paga impuestos).

 - Ejercicio 2 Escribe un programa que defina una clase Shape con un constructor que reciba como parámetros el ancho y alto. Define dos subclases; Triángulo y Rectángulo que hereden de Shape y que calculen respectivamente el área de la forma area().
